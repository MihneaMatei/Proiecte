Nume si prenume: Matei Mihnea-Cristian
Lucrare: Senzori de parcare

Descriere: Prezentul proiect reprezinta un sistem de senzori de parcare ce foloseste
un senzor ultrasonic de distanta ce trimite semnalul la o placuta Arduino. Semnalul este prelucrat, iar cu ajutorul unui buzzer este emis un semnal sonor a carui frecventa creste proportional cu scaderea distantei pana la obiect.
